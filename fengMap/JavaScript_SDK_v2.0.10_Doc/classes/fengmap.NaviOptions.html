    路径分析对象加载参数配置项

    名称类型标记描述
    [naviServerURL] String 可选的

    设置获取地图的服务器路径,默认是从蜂鸟视图服务器上获取地图

    示例:

    • 路径分析类参数配置
    //配置路径数据
var naviOpt = new fengmap.NaviOptions({
    mapServerURL:'data/scence' //直接从本地文件夹上获取地图数据，默认从fengmap服务器上获取。也可配置为其他服务器
});
