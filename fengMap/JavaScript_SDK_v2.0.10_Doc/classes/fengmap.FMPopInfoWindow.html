    控件参数配置

    名称类型标记描述
    IsScreenCoord Boolean

    控件参数配置,默认false，false时参数mapCoord地图坐标生效，true时screenCoord屏幕坐标生效

    mapCoord fengmap.FMMapCoord

    添加信息框的地图位置坐标，如：{x:1000,y:1000,groupID:1}

    screenCoord fengmap.FMScreenCoord

    添加信息框的屏幕位置坐标，如：{x:1000,y:1000}

    width Number

    信息框宽度

    height Number

    控件参数配置

    marginTop Number

    信息框距离地图的高度
    content String

    可为文本或html页面元素，如：'这是一个信息框'

    allLayer

    Unknown 只读

    当前楼层的状态,默认是单楼层的。
